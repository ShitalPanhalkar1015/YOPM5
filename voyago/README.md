# Voyago - Full-Stack Travel Booking Application

This project contains the frontend for the Voyago web application.

## Frontend

-   **Frameworks/Libraries:** HTML, CSS, JavaScript, Bootstrap 5
-   **Pages:**
    -   Home (Landing)
    -   Bus Booking
    -   Hotel Booking
    -   Package Deals
    -   Login & Register
    -   User Dashboard
-   **Features:**
    -   Responsive design for all devices.
    -   API integration with a separate Express.js backend.
    -   JWT-based authentication.
    -   Dynamic content rendering for bookings, packages, and search results.

## How to Run

1.  Open any `.html` file in the `client` directory in your web browser.
2.  Ensure the backend server is running on `http://localhost:5000` for API functionality.
